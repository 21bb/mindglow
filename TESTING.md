# 测试指南

## 快速开始测试

### 1. 设置环境变量（可选）

如果要测试 AI 分类功能，需要设置 Gemini API Key：

创建 `.env.local` 文件：
```
GEMINI_API_KEY=your_api_key_here
```

**注意**：如果没有设置 API Key，应用仍然可以运行，但分类功能会返回默认值。

### 2. 启动开发服务器

```bash
npm run dev
```

服务器将在 `http://localhost:3000` 启动。

## 常见问题

### Q: AI 分类不工作？
A: 检查是否设置了 `GEMINI_API_KEY` 环境变量，或查看浏览器控制台的错误信息。

### Q: 页面空白？
A: 检查浏览器控制台是否有错误，确认所有依赖已安装（`npm install`）。

### Q: 样式不正常？
A: 确认 Tailwind CSS CDN 正常加载，检查网络连接。
